# Achievement Notification Customizer: Remaining

- Screenshot capture, Steam screenshot triggering, notification image export, and screenshot-with-overlay generation.
- Full SteamAchievementNotifier theme import/export compatibility.
- Big Picture settings UI for editing every customizer field with controller navigation.
- Rich animation sequencing beyond existing Hydra notification animations.
